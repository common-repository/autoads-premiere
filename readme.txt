AutoAds Premiere allows you to quickly display advertisements on your website.

== Description ==

This plugin allows you to quickly add Google Adsense AutoAds to your website.

1. Install the plugin
2. Add your Publisher ID
3. Select pages to show adverts on
4. Determine roles to ignore
5. Save your settings
6. Profit!

In addition to specifying your Google AdSense Publisher ID you can also exclude specific roles from seeing advertisements with a quick click.

== Frequently Asked Questions ==
 
= Where do I find my Google Adsense Publisher ID =
 
Please visit [Google Adsense Publisher ID](https://support.google.com/adsense/answer/105516?hl=en)

= Advertisements are not showing =

* Be sure to include your full Publisher ID (Example: ca-pub-1234567890)
* Be sure to select pages and types to show advertisements on
* Be sure you did not choose NOT to show advertisements as Administrator while logged in as Administor
* Be sure to "view source" and make sure your code is showing in <head></head> tags
* If everything looks good, wait a few days for Google to decide where to place advertisements on your site (can take up to a week)
 
== Upgrade Notice ==
 
* Make sure you select pages and types of pages to show advertisements on or no advertisement code will render on your pages
